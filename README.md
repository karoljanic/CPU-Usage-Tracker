## CPU-Usage-Tracker

Multithreaded program for calculating CPU usage based on /proc/stat.
It presents producer-consumer problem.

**How to compile program:**
```sh
make
```

**How to clean build files:**
```sh
make clean
```

**How to run program:**
```sh
./main.out
```

**How to build tests program:**
```sh
make test
```

**How to run tests:**
```sh
./tests/test.out
```
